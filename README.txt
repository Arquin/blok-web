READ ME
Ik heb met HTML CSS en Javascript een prototype van 
een website gebouwd.
Dit is een concept voor een website waarmee de gebruiker een 
aantal verhalen kan lezen. Dit zijn verhalen voor onderweg. 
De gebruiker kan deze verhalen ook downloaden zodat hij ze 
ook zonder een internetverbinding kan lezen.

Ik heb CSS gebruikt om een grid aan te maken waar de item 
en verhalen geordend in staan die ook responsive is voor 
een smartphone en tablet. En met behulp van Javascript 
heb ik ook drie micro interacties in het prototype kunnen 
verwerken.
